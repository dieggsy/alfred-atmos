# Atmos #

Python port of Matlab's atmosisa


## Usage ##

- `atmos` — What gets shown…
    - `↩` or `⌘+<NUM>` — Action description…
        - `⌘+↩` — Alternate action description…
        - `⌥+↩` — Alternate action description…
        - `^+↩` — Alternate action description…
        - `⇧+↩` — Alternate action description…
        - `fn+↩` — Alternate action description…
    - `⇥` — Expansion description…
    - `⌘+C` — What gets copied…
    - `⌘+L` — What gets shown large…
